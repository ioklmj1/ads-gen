# Contributing to /ads-gen

Contributions are welcome. This document explains how.

## How to contribute

1. Fork this repository
2. Create a branch for your change
3. Make your changes
4. Submit a Pull Request with a clear description of what you changed and why

## What makes a good contribution

- Bug fixes in the orchestration logic
- New QC gate criteria based on production experience
- Support for additional media generation APIs
- Documentation improvements
- Example productions (storyboards, briefs, outputs)
